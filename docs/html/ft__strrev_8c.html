char* ft_strrev ( const char *  s)

Reverses a string.

Allocates (with malloc) and returns a “reversed” NULL-terminated string.

Example
ft_strrev("0123456789") returns "9876543210"
Parameters
sString to be reversed.
Returns
Reversed string or NULL if allocation failed.
